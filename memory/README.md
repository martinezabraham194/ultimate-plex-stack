# Project Memory

This directory contains the persistent memory for the project, documenting the context, decisions, and evolution of the codebase.

## Files

- **initialization.md**: Initial project context, file discovery, and assumptions.
- **decisions.md**: Key design decisions, rationale, and dates.
- **system_patterns.md**: Architecture, design patterns, and component relationships.
- **changelog.md**: High-level summary of major plan changes/fixes.
- **failure_log.md**: Indexed database of errors encountered.
- **solutions.md**: Indexed database of proven fixes.
